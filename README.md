## SaleGameOnline

Sale Game Online is project use mongoDB run on NodeJS 

## Installation

install MongoDB in your computer

install project
```sh
$ git clone https://github.com/kittanet/SaleGameOnline.git
```

and install node
```sh
$ npm install
```

run mongoDB
```sh
$ mongod
```

run app
```sh
$ node app.js
```
